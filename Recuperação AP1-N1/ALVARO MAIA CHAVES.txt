Tem sido fundamental pois, com a grande quantidade de dados foi possível aumentar a acurácia de modelos treinados por conta da disponibilidade de mais detalhes de dados. E com o processamento aprimorado, aumentou ainda mais a velocidade e eficiência de que esses dados são processados pelo o modelo e usados para treina-los.
b) 
Um exemplo é reconhecimento por imagem, porque com a facilidade de obter esses tipos de dados e de processá-los ajudou a melhorar a eficácia e acurácia desses tipos de aplicações. Outra exemplo de aplicação é na área de marketing, onde ficou extremamente fácil obter dados dos consumidores e processar essa grande quantidade para recomendar produtos ou serviços mais eficientemente para o gosto de cada consumidor.
c) 
Conjunto de treinamento: Conjunto rotulado de dados que será usado para treinar o modelo. Conjunto de validação/teste: São novos dados, diferentes do de teste, servindo para ver se o modelo pode criar hipóteses corretas sobre eles. Rótulos: São dados de entrada que são rotulados para ter uma saída esperada, como a entrada ser a imagem de um gato e a saída ser identificada como um gato. Variáveis Independentes do modelo: São as saídas do modelo que são produzidas através da analise dos dados de entrada. Modelo: É o estado inicial (Preparando os dados) ou final (Conjunto de treino  já usado e o modelo validado) do aprendizado de maquina, em resumo o objetivo para oque se usara o aprendizado de maquina. Exatidão do modelo: É a forma de que se quantifica a acurácia do modelo acerca de interpretar os dados corretamente, sendo assim quanto maior melhor. Hold-Out 80/20: É uma forma validar um modelo quando se há mais acertos do que erros. Cross-Validation: É quando se verifica manualmente se as saídas do modelo condizem com o esperado. 
d) 
Classificação tenta prever  saídas rotuladas discretas ou seja, a partir de uma entrada, certa saída pode ocorrer, geralmente sendo uma classe, exemplo seria se um paciente se encontra em uma classe saudável ou doente usando os dados de entrada oferecidos. Já a regressão serve para prever saídas continuas ou seja através da analise de vários atributos juntos, uma saída deve ocorrer, geralmente sendo saídas racionais, um exemplo de regressão é o preço de um terreno considerando para o preço a área, localização, condição, etc...
